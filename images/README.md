# Images Folder

Add your images here:

## Required Images

| File | Size | Description |
|------|------|-------------|
| `logo.png` | 180x180 | App logo (already added) |
| `favicon.png` | 32x32 | Browser favicon |
| `app-preview.png` | 300x600 | Main app screenshot for hero section |
| `og-image.png` | 1200x630 | Social media share image |

## Screenshots Folder

Add app screenshots to `/screenshots/`:

| File | Size | Description |
|------|------|-------------|
| `splash.png` | 220x450 | Splash screen |
| `login.png` | 220x450 | Login screen |
| `dashboard.png` | 220x450 | Dashboard/Home screen |
| `ai-input.png` | 220x450 | AI input screen |
| `tasks.png` | 220x450 | Tasks screen |

## Tips

- Use PNG format for best quality
- Keep file sizes under 500KB for fast loading
- Screenshots should have rounded corners matching the mockup style
